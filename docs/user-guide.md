# Pic Collage Maker — User Guide

## Quick Start

### English
1. **Add photos**: Tap the "Photos" tab and select images from your device
2. **Arrange**: Drag to move, pinch/scroll to zoom the canvas. Select an element to resize or rotate.
3. **Style**: Use the "Filters" tab for photo effects, "Background" for board styling, "Text" and "Stickers" for decorations.
4. **Export**: Tap "Export" in the header to save as PNG/JPG or share directly.
5. **Save project**: Use "Projects" menu to save your work and return later.

### Deutsch
1. **Fotos hinzufügen**: Tippe auf den "Fotos"-Tab und wähle Bilder aus.
2. **Anordnen**: Ziehen zum Verschieben, Zoomen mit Scroll/Pinch. Element auswählen für Größe/Drehung.
3. **Gestalten**: "Filter" für Effekte, "Hintergrund" für Stil, "Text" und "Sticker" für Deko.
4. **Exportieren**: "Export" in der Kopfzeile für PNG/JPG oder Teilen.
5. **Projekt speichern**: "Projekte"-Menü zum Speichern und späteren Weiterarbeiten.

## Features

### Projects
- Save multiple projects locally
- Auto-save every 1.5 seconds while editing
- Export as `.piccollage` file to transfer between devices

### Layers
- Reorder with "Bring forward" / "Send backward"
- Hide/lock elements
- Adjust opacity (0-100%)
- Blend modes: Normal, Multiply, Screen, Overlay, Darken, Lighten

### Export Options
- **PNG**: Lossless, best quality
- **JPG**: Smaller file, adjustable quality
- **WebP**: Modern format (where supported)
- **Platform presets**: Instagram square/portrait/story, A4

## Tips
- Use grid layouts for quick, evenly-spaced collages
- Tap a photo then "Crop & Shape" to clip into circles, stars, hearts
- Double-tap text to edit inline
- Undo/redo with Ctrl/Cmd+Z and Shift+Z

## Privacy
Everything stays on your device. No photos are uploaded anywhere. Projects are stored in your browser's IndexedDB.
